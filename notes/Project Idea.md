# Project Idea
## Description
An ESP-IDF project. Something along the lines of a "Wi-Fi Signal Dashboard Appliance."
- The ESP32 connects to the user's home Wi-Fi
- Measures signal quality
- Serves a dashboard 
### ESP-IDF as a Framework
It already gives us what we need
A Wi-Fi driver
- Reports RSSI
- Scan nearby access points
- Built-in HTTP server
